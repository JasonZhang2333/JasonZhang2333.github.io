### 思想篇

###### 邱国鹭：投资的本质三个维度

**行业的维度**

- 行业商业模式：白酒行业毛利80%到90%
- 行业竞争格局：空调行业双龙头
- 行业空间：相比夕阳行业和初创行业，成长期和稳定期的行业特别好投资，白电在稳定器，有些消费品在成长期
- 行业门槛：要想清楚一家公司为什么能够保持较高的资本回报率和净利润率，资源独占、牌照限制、技术优势、品牌优势

**公司的维度**

看公司管理机制和产品定位，问自己公司如果继续发展，是越大越强还是越大越难，有没有规模优势

**管理层的维度**

- 战略能力
- 战术执行力

###### 邓晓峰：中国价值投资思路

- 价值投资要看估值水平，价值投资不等于长期持有，长期持有是一个结果而不是目的
- 价值投资要面向未来，找到持续创造高回报的企业，分享企业的价值增量

- 市场在不断变化，中国市场越来越有效，超越指数越来越困难
- 学会“Think Different”，不断拷问自己去学习，要考察自己的偏见，像对夕阳产业、国有企业的偏见

要立足长远视野，把对企业长期发展的判断，作为超额收益的立足点，中国这样巨大的经济体给了我们这样的机会

###### 卓利伟：我理解的基本面投资

市场存在众多变量，难以把握，要降维，降到3个变量：
- 企业价值判断
- 产业趋势判断
- 系统性风险水平

基本面研究是广泛研究，把复杂市场降维为几个长期最重要的变量，在少数领域做出含金量最高的决策，提高投资的收益，这一过程可以不断复制和自我积累。

###### 孙庆瑞：投资行业周期

- 自上而下从宏观经济和产业发展角度圈定行业
1. 结构变迁：受益于经济发展不断壮大的行业
2. 周期轮回：当前经济周期最受益的行业
- 自下而上从人和机制的角度挑选公司

**成长期行业**

发掘强需求：轻工->重工->娱乐->健康环保

成长行业，创新能力强、敢闯敢干的企业家更可能胜出

爆发初期PE高没关系，增长强，中后期一定要看PE，看行业供给情况是否过剩

从自己的需求出发类推全社会

**稳定期行业**

需求趋于稳定，强者恒强，通过提升份额和利润率来增长

**看好的行业**
- 未来依然是结构性行情
- 消费升级：消费品、运动、健康
- 制造业

###### 冯柳：逆向投资

逆向投资核心是在负面思考时买入，负面被显著体现的时候，正面解读便可展开，我们挣的是市场角度和变化的钱，但要避免负面被正确展开的过程

市场永远正确，每次涨跌都是一个角度和思维的过程，市场是集体智慧的体现，可以帮助我们完善视角

集中投资的核心是确定性，一致的确定性会降低赔率，当确定性和赔率统一的时候可以重仓

选股核心，前两条决定业绩和持续性，最后一条决定能否在估值提高的情况下表达业绩：
1. 可预期：一年内的业绩和估值
2. 可展望：未来三年的发展路径
3. 可想象：十年后要有想头

- 战略性投资：买热点龙头，贵点没关系
- 战术性投资：买有安全边际的基本面不差的企业，等待变化

抄底：
- 杀估值：最好，跌下来下跌逻辑就没了
- 杀业绩：可以针对经营节奏和变化投资
- 杀逻辑：远离

不控制市场风险：
- 很多市场风险可以通过选股化解
- 可以降低投资复杂度，大多数人不具备挑战复杂的能力，可以把市场风险当做理所应当，做好个股

市场不认可选择的股票，结善缘开恶果并不少见，可以有提醒和纠错的机制，和耐心持股的区别在有有没有意外和不理解的地方

博弈论选股：
- 关注度：来源于报告和交流热度
- 购买度：图形阶段和筹码结构
1. 高关注度低购买度：估值容易抬升
2. 高关注度高购买度或低卖出度：估值有下行压力
3. 低关注度高购买度或低卖出度：估值虽低但缓慢提升
4. 低关注度低购买度：估值不容易有大变化

投资要听三个声音：
1. 企业的声音
2. 市场的声音
3. 内心的声音

主仓事关重大，长期持有有各种挑战，极端情况会放大内心的缺点，通过增减仓不断假设和变换角度，倾听和体会自己，学会引导和利用自己的情绪

###### 王世宏：投资经济变化主航道

总结：
- **创新**诞生新生意或者整合原有生意，实现高增长
- 拥有**护城河**的好生意能持续实现高增长，带来高资本回报率
- 创业者推动创新出现，并创造好的组织和商业模式加深护城河，为投资者创造价值
- 合适的价格带来较高的内部收益率

产业升级路径：新产品的规模化生产导致价格降低和居民收入提升，收入提升到一定程度迎来产品需求爆发，这种需求爆发按照消费升级过程逐渐产生

不同类型生意：
超一流生意：高增长、低资本投入、规模经济护城河带来高资本回报，如互联网
一流生意：中等增长、低资本投入、品牌护城河带来中高资本回报，如白酒
中等偏上生意：中等增长、高资本投入、中等规模经济护城河带来中等资本回报，如银行、地产
差的生意：没有护城河导致的低资本回报生意

公司分析步骤：
1. 商业模式
2. 供需关系
3. 供应链上竞争力
4. 不同假设下公司的发展

**创新是价值创造的基础**

创新标准：
- 产品或服务体验完全不同
- 效率完全不同

创新带来新生意：
1. 动力提供方：公用事业生意
2. 材料生产方：重资本
3. 动力机器：技术密集
4. 具体应用：轻资本消费品

当前主航道：**互联网**
1. 电商
2. LBS（位置服务）公司：滴滴、美团、Uber、Airbnb
3. feed流：社交、今日头条
4. feed视频流：快手、抖音

**护城河是价值实现的基础**

好生意：
- 网络效应
- 品牌效应
- 产品粘性
- 规模经济

用户边际收益递增需要网络效应和品牌效应，企业端边际成本递减需要规模经济

- 20世纪50年代之前，规模经济时代：好生意是具有规模经济的生意，在市净率低时买入
- 1950年到1990年，品牌时代：由于电视等现代化渠道的出现和规模经济，好生意是品牌的生意，如以品牌作为护城河的消费品生意和以专利作为护城河的医药生意，寻找持久的品牌，在价格除以现金流贴现结果低时买入
- 1990年以来，网络时代：好生意是具有网络效应的生意，护城河是建立产品和用户之间的粘性，尤其是用户和用户之间的联系网络，寻找可以够坚强大网络效应的生意，在用户扩张早期买入，价格除以现金流贴现结果依然适用

组织的扩张产生规模经济：
- 第一次扩张：亚当斯密时代的分工和专业化
- 第二次扩张：福特汽车发明的流水线作业
- 第三次扩张：沃尔玛和福特的柔性作业
- 第四次扩张：Office和ERP等软件的大规模使用
- 第五次扩张：互联网和移动互联网的应用
- 第六次扩张：人工智能对人的替代

未来产业分析：
1. 是否可以接触客户
2. 接触中是否产生数据，是否可以分析应用数据
3. 是否能够产生个性化产品或服务
4. 是否可以根据每个人的反馈优化产品或服务

**坚持投资经济主航道**

中国主航道：
- 2003年之前：基础设施，如港口、机场、电力和移动网络等
- 2004年以后：消费品和医疗，如白酒、白电、医药等
- 2010年之前：PC互联网时代
- 2010年之后：移动互联网时代

判断新生意能否成为主航道生意：
- 体验或者效率有数倍提升
- 足够规模
- 足够增速
- 商业模式可行
- 有一批消费者使用体验并引领

成熟生意是否还在主航道上：
- 渗透率是否足够高
- 增速是否放慢

新航道：
- 流媒体
- 云计算
- 娱乐
- 互联网金融
- 新能源汽车
- 人工智能
- 教育
- AR/VR
- 区块链
- 生物医疗
- 自主汽车品牌、核心电子元件、创新药等中国在全球份额迅速扩大的领域

###### 卓利伟：把最重要的事做到极致

**研究最重要的问题**
- 时间是稀缺资产，要研究最重要的问题：要研究大问题，把握产业大方向，决策获得大概率的成功
- 在正确的道路上持续积累：树立科学合理正确的研究方法，持续优化完善思维框架和研究体系

**四个维度思考什么是重要问题**
- 宏观逻辑：宏观经济、社会思潮、群体心理、政治经济关系等大格局
- 产业逻辑：不同产业不同发展阶段，核心驱动因素和竞争要素是不同的
- 业务逻辑：看企业战略布局、业务方向是否符合宏观逻辑和产业逻辑，是否符合社会大潮和产业大方向
- 财务逻辑：分析验证企业业务上的特征和问题

**抓住产业本质、核心要素、关键变化**
- 不同产业本质在经济学和商业模式上有显著特点：水电行业是加了杠杆的利率产品，传统零售接近于商业地产的租赁业务
- 要研究驱动产业发展的核心要素：产品力驱动消费品行业
- 要把握拐点时刻的关键变化：消费品行业人口红利基本消失，大单品增长的时代结束，存量的结构升级和中小规模的品类创新

**看企业核心能力**
- 时间：好的商业模式顺应社会发展大势，为社会贡献福利，这样的商业模式才是持久的，是时间的朋友
- 空间：好的商业模式要规模经济，例如餐饮业依赖厨师规模不经济难以扩张
- 人：企业家精神，企业的核心竞争力实际上是公司治理结构，这是企业家和核心团队的思想成果

**消费品行业**
1. 宏观逻辑和产业逻辑
- 人口红利基本消失
- 中产阶级人口和互联网一代人口占比提高：体验、服务、健康、审美等需求将成为购买决策的重要因素
- 从过去的企业导向到消费者导向，利用新技术建立与消费者的高效沟通，实现需求的精准对接
- 互联网消除了信息不对称和渠道差异，产品创新、有效传播、与消费者良性互动更为重要
2. 业务逻辑和财务逻辑：会看老品类毛利率、新品类占比和边际利润、市场份额、客户量、客单价、复购率等多个维度，以验证企业是否具有持续的学习和创新能力

**服务业**
1. 宏观逻辑：体验消费和服务消费占比提升，商品和服务相融合
2. 产业逻辑：站在用户角度，建立更好的体验和口碑
3. 业务逻辑和财务逻辑
- 用户数量变化
- 用户续单率和ARPU，是对口碑和体验的最佳验证
- 新客户的边际成本和边际利润
- 客户的学习成本是否足够低，转换成本是否足够高

**制造业**
1. 宏观逻辑
- 中国制造业难被替代
- 中国仍有工程师人口红利
- 中国制造业背靠庞大市场，试错成本低
- 发达国家经济低迷，制造业企业财务状况差，给了中国企业走出去的机会
2. 产业逻辑
- 传统制造业的核心逻辑是产品和流程的标准化、规模化，达成时间和空间上的自我复制能力
- 数据与计算能力的发展，使得个性化量产成为可能，比如定制家具、制衣
3. 业务逻辑和财务逻辑
- 广义产能规模的扩张能否带来正的边际利润，能否提升公司整体的资产回报率水平
- 规模扩张能否带来长期竞争力的提升，如核心技术和品牌积累

**TMT行业**
1. 宏观逻辑
- 技术进步是长期增长潜力的重要因素
- 技术进步速度超出宏观总需求增速（存疑），初次分配集中到技术精英和资本身上
2. 产业逻辑：技术进步的演进路径、技术成果的商用普及以及由此带来的商业模式的创新路径尤为重要，同一细分行业不同公司的命运天壤之别
3. 业务逻辑：技术路径选择错误的成本非常高，主流技术所引导的全产业链研究尤为重要
4. 财物逻辑：技术导向型公司的收入、利润、现金流变化不像传统制造业公司
- 技术研发型企业，最先增加技术人员与研发费用，然后是收入，最后是净利润和现金流
- 商业模式创新的服务型公司，最先增加的是流量与用户，然后是收入的增长，最后是净利润和现金流

研究的三个环节：
1. 归纳：整合信息，搞清楚问题的逻辑关系和主要矛盾
2. 演绎：提出几种最可能的假设，需要创造性思维和想象力
3. 实证：在实证中启发新思维

优秀研究员特质：
1. 求知欲/好奇心：无任务下的自我驱动力
2. 诚实：自省，纠错，心无偏见
3. 独立：独立思考，独立判断

日常工作建议：
1. 建立属于自己的智慧圈：多认识行家朋友，交流思想，互相启迪
2. 广泛学习与阅读
3. 构建属于自己的研究体系
4. 结构化研究文档：记录在Excel里，数字化表达，总结阶段性思考

### 实战篇

###### 邱国鹭：投资中的变与不变

市场中唯一不变的就是变化本身，要看变化是长期变化还是短期变化，例如很多优质公司遇到了短期困难就是买入良机，要看变化是结构性变化还是周期性变化，周期性变化是短期冲高，结构性变化是长期变化，高毅资本更关注结构性变化，例如汽车行业中自主品牌的崛起、定制家具的产业链再造等等。

目前市场发生了以下几种变化：
1. 市场环境变化，一些优秀的企业成长起来
2. 投资者的变化，一些有着十几年的优秀业绩和长期价值投资理念的投资人
3. 监管的变化，以扶植价值投资、长期投资为导向
4. 客户导向的变化，认同价值投资，愿做长期投资

投资中的不变：
1. 便宜才是硬道理
2. 找行业格局清晰、相对容易研究的行业
3. 研究透商业模式、行业属性
4. 逆向思维和独立思考，投资机会常出现在被市场冷落的角落

###### 邓晓峰：基本面投资中国实践

**市场不够有效，建立自己的坐标系**

市场本身真正反映的是国家、产业发展的变化，要从基本面这个角度投资。

基金经理除了要了解公司、股市规律和其他参与者的行为，还要理解经济运行的一般规律，可以使你有信心做出独立决策。

美国市场越来越有效，驱动着投资人不断努力，中国同美国有些差异：
1. 参与者众多，很多不专业，导致市场不那么有效
2. 机构中的公司治理和考核机制导致的扭曲

市场无效可能会阶段性出现，例如13-15年传统行业因需求放缓等因素进入调整阶段，互联网行业却在飞速发展，大家容易忽视比较稳定的传统行业，在16、17年互联网对各个行业影响高度明确化，不确定性下降之后，低估行业估值被修复。

资本市场各阶段有各阶段的特点，九十年代到03年庄户横行，03年到06年关注企业价值，07年之后大股东逐渐扭曲资本市场，未来一定是选择优秀的公司，可以将美国市场作为坐标系参考。标普500中，金融行业最高到了22%，12年是16%，医疗行业也很重要，高科技行业占比最大，12年占到18%，现在大概是24%到25%，可选消费和日常消费比较稳定，大概10%，制造业占GDP大概10%，还很有生命力和竞争力。

估值方面，医疗行业长期市盈率较高，电信和公用事业由于美国利率走低稳定行业估值较高，金融行业市盈率较低，因为杠杆较高有较高的风险溢价，科技行业利润占比高于市值占比，可能是不确定性较高，有较高的风险溢价，但是科技行业近年一直保持高增长，消费品行业估值和市场整体相近，这些都可以作为中国市场的参考。

**案例一 通信行业的兴起**

通信行业人力资本密集，研发人员占比超过1/3，毛利30%-50%，研发费用10%-15%，销售费用10%，中国通信行业依靠人力成本优势打败了国际巨头。

**案例二 市场误解中国银行业**

市场对中国银行业有许多负面看法，如政策保护导致高利润、坏账很多等：
- 中国息差处于中间水平，从GDP增速来看中国息差还偏低，银行高利润主要来自劳动生产率的提高，中国银行人力成本低
- 日本金融危机后处理的坏账占贷款的18%，温州的金融危机也大体是18%，2016年中国的银行业确认了6.5%的贷款坏账，坏账已经过去大半

银行业目前处于工业化时期的对公业务为主到向零售、消费贷转型的过程。

**未来的机会**

从美国股市历年的前十大公司来看，主要分布在互联网、消费品、医药、金融等行业，本质都是ToC的行业，中国的发展一定会带来劳动力成本的上涨，未来的行业回报率会发生变化，标准化产品回报低、个性化产品回报高，品牌会有溢价，未来行业关注市占率。

###### 邓晓峰：市场认知偏差中寻找机会

研究体会：
1. 研究增长型、体量大的行业，投入产出比高
2. 研究要由少到多，再由多到少，先多了解行业资料、发展历史、发达国家经验、我国所处阶段、相关企业行业传记等，建立对行业的总体认识；之后再逐渐抽象总结，把对行业的理解提炼成一些指标和参数预测行业的发展，不停修正，**先把行业变厚，再把研究变薄**
3. 长期跟踪优秀公司，对优秀公司的长期跟踪是行业研究的捷径
4. 问题是研究的方向，要寻找市场的分歧，发现市场的偏差
